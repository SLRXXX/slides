Canvas Basic

  • Immediate mode API

    Call rendering commands to draw the entire scene

  • Currently 1.0
  • IE9+, Chrome, Firefox, Safari...
  • Coming 2.0

Create Context

var canvas = document.createElement('canvas');
canvas.width = 500;
canvas.height = 500;
var ctx = canvas.getContext('2d');
  • Path
  • Image
  • Text

Path

  • Line
  • Bezier Curve(Cubic, Quadratic)
  • Arc

Draw a line

ctx.beginPath();
ctx.moveTo(100, 100);
ctx.lineTo(500, 100);
ctx.stroke();

Draw a Cubic Bezier

ctx.beginPath();
ctx.moveTo(100, 100);
ctx.bezierCurveTo(300, 100, 200, 300, 500, 300);
ctx.stroke();

Draw a Quadratic Bezier

ctx.beginPath();
ctx.moveTo(100, 100);
ctx.quadraticCurveTo(300, 100, 500, 300);
ctx.stroke();

Draw a Circle

ctx.beginPath();
ctx.arc(300, 200, 100, 0, Math.PI * 2, true);
ctx.stroke();

Smile

ctx.beginPath();
ctx.arc(300, 180, 100, 0.2, 3, false);

ctx.stroke();

Styling Path

  • fillStyle

    - ctx.fillStyle = '#fff', CanvasGradient, CanvasPattern

  • strokeStyle

    - ctx.strokeStyle = '#fff', CanvasGradient, CanvasPattern

  • Line Style

    -ctx.lineWidth = 2;
    -ctx.lineCap = 'butt', 'round', 'square';
    -ctx.lineJoin = 'round', 'bevel';

  • Shadow

    -ctx.shadowColor = '#000';
    -ctx.shadowBlur = 5;
    -ctx.shadowOffsetX = 40;
    -ctx.shadowOffsetY = 40;

Image

ctx.drawImage(image, 0, 0, image.width, image.height);

Text

ctx.fontStyle = '30px Helvetica Neue';
ctx.strokeText('蛤蛤蛤蛤', 30, 30);
